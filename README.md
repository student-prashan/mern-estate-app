# MERN Real Estate Marketplace

A full stack real estate marketplace application built using the MERN stack.

## Tech Stack

Frontend
React
Tailwind CSS

Backend
Node.js
Express.js

Database
MongoDB

Authentication
JWT

## Features

User authentication and login
Create and manage property listings
Search and filter properties
Image uploads for listings
User dashboard for managing properties
RESTful API architecture
